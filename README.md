Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @anacldrn/react-repository`

## Usage

    var numFormatter = require('@anacldrn/react-repository');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests
